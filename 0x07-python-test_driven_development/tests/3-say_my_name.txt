This library is very simple, since it only has one function called
``say_my_name()``.

::

    >>> say_my_name = __import__('3-say_my_name').say_my_name

::

    >>> say_my_name("Mdu", "Harvey")

    >>> say_my_name("Sipho")
    Sipho
